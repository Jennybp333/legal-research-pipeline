# 사내 톤 가이드 (Internal Memo Tone Guide)

이 가이드는 **사내 법률팀 내부 회람용 메모**에 적용됩니다.

---

## 기본 원칙

| 항목 | 사내용 |
|---|---|
| 어조 | 평어체, 사실 진술 위주 (`~이다`, `~로 본다`) |
| 결론 위치 | **결론 먼저(BLUF: Bottom Line Up Front)** → 근거는 그 다음 |
| 분량 | 1~2페이지. 본문 600~1200자 |
| 약어 | 자유롭게 사용 (`민법`, `상법`, `대판` 등) |
| 인용 | 인라인 약식 인용 OK (`(대판 2020다12345)`) |
| 면책 | "변호사 최종 검수 필요" 한 줄로 충분 |

## 구조 (권장)

```markdown
# [사내 메모] <케이스 ID 또는 한 줄 제목>
작성: <작성일자> / 의뢰: <팀명>

## 결론
<2~4 문장. 질문에 대한 직접 답변>

## 핵심 근거
1. <쟁점 1 + 근거 + 인용>
2. <쟁점 2 + 근거 + 인용>
3. <쟁점 3 + 근거 + 인용>

## 리스크 및 추가 검토 필요 사항
- <Critic 노트에서 도출된 반대 논거, 약한 추론>

## 후속 액션
- [ ] <팀이 결정해야 할 사항>

---
※ 본 메모는 변호사 최종 검수 전 초안입니다.
```

## 어휘 / 표현 가이드

- **OK**: "위반 가능성이 높다", "쟁점은 ~이다", "판례 흐름상 ~"
- **피할 것**: "~해야만 합니다", "절대로", "당연히" (단정 어조)

## 사내용에서 **생략 가능**한 것

- 법률 용어 해설 (수신자가 법무팀)
- 사실관계 장황한 재진술 (핵심만 짧게)
- 정중한 도입부 ("귀사의 문의에 대해 답변드립니다" 등)

## 사내용에서 **반드시 포함**할 것

- 결론을 한눈에 파악할 수 있는 BLUF
- 반대 논거 / 리스크 (Critic 단계에서 식별된 것)
- 추가 검토가 필요한 항목 (변호사 판단이 필요한 영역 명시)
