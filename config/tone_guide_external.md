# 외부 톤 가이드 (External Opinion Tone Guide)

이 가이드는 **외부 클라이언트에게 제공하는 법률 의견서**에 적용됩니다.

---

## 기본 원칙

| 항목 | 외부용 |
|---|---|
| 어조 | 정중한 경어체 (`~입니다`, `~로 사료됩니다`, `~로 판단됩니다`) |
| 결론 위치 | **사실관계 → 쟁점 정리 → 법령·판례 검토 → 결론** (전통 의견서 구조) |
| 분량 | 2~5페이지. 본문 1500~3500자 |
| 약어 | 첫 등장 시 풀어서 표기 (`민법 제103조(반사회질서 법률행위)`) |
| 인용 | 정식 인용 (`대법원 2020. 3. 26. 선고 2019다12345 판결`) |
| 면책 | 별도 단락의 정식 면책 조항 |

## 구조 (권장)

```markdown
# 법률 의견서

**의뢰자:** <클라이언트 회사명>
**작성일자:** <YYYY년 MM월 DD일>
**사건명/케이스 ID:** <식별자>

## 1. 의뢰 요지
귀사로부터 <간략 의뢰 요지>에 관하여 법률 의견을 요청받았습니다.

## 2. 사실관계
귀사로부터 제공받은 자료에 따른 사실관계는 다음과 같습니다.
1. ...
2. ...

## 3. 법적 쟁점
본 건의 주요 법적 쟁점은 다음과 같이 정리됩니다.
가. <쟁점 1>
나. <쟁점 2>

## 4. 검토
### 가. <쟁점 1>에 관하여
관련 법령 및 판례를 검토한 결과는 다음과 같습니다.
- <법령 인용 + 풀어 쓴 해설>
- <판례 인용 + 사실관계 적합성 분석>

### 나. <쟁점 2>에 관하여
...

## 5. 결론
이상의 검토를 종합하면, ...로 사료됩니다.

다만, 본 의견은 아래 사항에 유의하시기 바랍니다.
- <리스크 / 추가 검토 필요 사항>

## 6. 면책 조항
본 의견서는 작성일 현재의 관계 법령 및 판례를 기준으로 작성되었으며,
귀사로부터 제공받은 사실관계가 정확하다는 전제하에 작성된 것입니다.
사실관계의 변경이나 법령 개정이 있는 경우 결론이 달라질 수 있으며,
본 의견서는 일반적 법률 검토 자료로서 특정 거래의 최종 의사결정에 사용하시기 전에
당사 담당 변호사와 별도 협의가 필요합니다.

작성: <법인명 / 변호사명>
```

## 어휘 / 표현 가이드

- **OK**: "~로 사료됩니다", "~로 판단됩니다", "~할 여지가 있습니다", "유의가 필요합니다"
- **피할 것**: "당연히", "분명히", "절대로", "확실히" — 단정형은 의견서 책임 리스크
- **약어 금지** (또는 첫 등장 시 풀어 쓰기): `대판` → `대법원 판결`, `상법` → `상법(법률 제xxx호)`

## 외부용에서 **반드시 포함**할 것

- 사실관계 명시 (귀사로부터 받은 사실에 기초함을 명시)
- 법령·판례의 정식 인용
- 별도 단락의 면책 조항
- 변호사 협의 권고 문구

## 외부용에서 **피해야** 할 것

- 약어, 은어, 사내 농담
- 결론 단정 ("...은 위법이다")
- 의뢰자의 의도 추측
- 사내 메모에서 사용한 BLUF 구조 (외부용은 전통적 4단 구조)
